# OOP_lab7

Программа-симуляция подземелья с NPC (рыцари, эльфы, друиды). Реализована многопоточная логика, наблюдатели, сохранение/загрузка и тесты.

Содержимое:
- исходники: .cpp/.h (DungeonEditor, FightVisitor, Factory, NPC, Observer, main)
- CMakeLists.txt — сборка проекта и юнит-тестов с использованием GoogleTest
- tests/ — тесты на GoogleTest

Требования
- CMake >= 3.14
- Компилятор с поддержкой C++17
- Сеть (CMake скачает GoogleTest через FetchContent) — либо GoogleTest доступен локально

Сборка и запуск
1. Создайте каталог сборки и перейдите в него:
   mkdir build
   cd build

2. Сгенерируйте сборку с помощью CMake:
   cmake ..

3. Постройте проект:
   make

4. Запустите программу:
   ./dungeon_main

   Программа запустит симуляцию (30 секунд) и выведет список выживших, а также сохранит файл dungeon_save.txt.

Запуск тестов
После сборки выполните:
   ./unit_tests

Файлы логов и сохранений
- log.txt — лог событий боёв 
- dungeon_save.txt — результат saveToFile
